# 🐍 SERPENT-X // NEURAL ARCADE

A futuristic cyberpunk-inspired Snake game built with pure HTML, CSS, and JavaScript.

Experience the classic Snake gameplay reimagined with neon visuals, glowing effects, animated particles, level progression, bonus collectibles, and a sci-fi user interface.

## 🚀 Live Demo

Play Now:
https://rahulkattamuri2007-cmd.github.io/futuristic-snake-game/

---

## ✨ Features

### 🎮 Gameplay

* Classic Snake mechanics
* Progressive difficulty system
* Level-based speed increases
* Score and High Score tracking
* Bonus collectibles for extra points
* Collision detection and game over system

### 🎨 Visual Effects

* Cyberpunk-inspired UI
* Animated starfield background
* Neon glow effects
* Particle burst animations
* Dynamic food animations
* Futuristic HUD display
* Scanline and vignette effects

### 📱 Accessibility

* Keyboard controls
* Touch swipe support for mobile devices
* Responsive design
* Local storage support for saving high scores

---

## 🎯 Controls

| Action         | Keys  |
| -------------- | ----- |
| Move Up        | W / ↑ |
| Move Down      | S / ↓ |
| Move Left      | A / ← |
| Move Right     | D / → |
| Pause / Resume | P     |

## 🛠️ Technologies Used

* HTML5 Canvas
* CSS3 Animations
* Vanilla JavaScript (ES6)
* Local Storage API

No frameworks or external game engines were used.
